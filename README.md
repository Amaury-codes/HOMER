H.O.M.E.R. — High-Order Metadata Engine & Repository

A desktop application for document indexing operators.
Built with Python, PyWebView and a lightweight HTML/JS frontend.

Features
- Integrated PDF viewer with page rotation, deletion and reordering
- Configurable index fields (text, date, dropdown)
- Multi-client profile management
- Classification plans and sub-plans
- Database lookup via Excel reference file
- Normalized TXT export companion file
- Indexing statistics with CSV export
- Auto-update via GitHub Releases
